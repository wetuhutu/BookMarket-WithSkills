# 开发日志一

今天开始做这个校园二手书交易平台的项目

先确定了技术栈，前端用Vue 3，后端用Spring Boot + MyBatis，数据库用MySQL。选这些主要是考虑到Vue 3的Composition API写起来比较方便，Spring Boot生态成熟，MyBatis对SQL控制比较灵活。

然后搭了项目的基础结构。前端用Vite创建项目，配置了Vue Router做路由，Pinia做状态管理，Tailwind CSS做样式。后端创建Spring Boot项目，配置了MyBatis和数据库连接。

数据库设计了几个核心表：用户表、书籍表、分类表、订单表、收藏表、评价表。用户表存用户基本信息和卖家信息，书籍表存书籍详情和卖家ID，分类表存书籍分类，订单表存交易记录，收藏表存用户收藏的书籍，评价表存书籍评价。

设计API接口的时候，参考了RESTful风格，设计了用户管理、书籍管理、订单管理、收藏管理等几类接口。每个接口都定义了请求方法、路径、参数和响应格式。

今天遇到的问题是Spring Boot版本和MyBatis版本不兼容，启动的时候报错。查了一下，发现是MyBatis的版本太新了，不兼容Spring Boot 3.5.9。最后把MyBatis降级到3.0.3就解决了。

明天打算开始写核心功能，先做用户认证模块，包括注册、登录、JWT token生成这些。
